Environment
===========

This project requires the following Vite environment variables to be set in a `.env` file at the project root for Supabase integration:

- `VITE_SUPABASE_URL` — your Supabase project URL
- `VITE_SUPABASE_ANON_KEY` — your Supabase anon/public API key

A sample `.env` with placeholder values has been added. Replace the placeholders with your real values and avoid committing secrets for production.
